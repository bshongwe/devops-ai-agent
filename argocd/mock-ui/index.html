Setup Real ArgoCD

To deploy with real ArgoCD in a Kubernetes cluster, use the configuration files in the argocd/ directory:

kubectl apply -f argocd/application.yaml

Or install ArgoCD in your cluster:

kubectl create namespace argocd
kubectl apply -n argocd -f https://raw.githubusercontent.com/argoproj/argo-cd/stable/manifests/install.yaml

CI-CD Agent Integration

The CI-CD Agent can trigger ArgoCD deployments through:

  • API Integration: Direct ArgoCD API calls for application sync
  • GitOps Workflow: Push changes to Git repos monitored by ArgoCD
  • Webhook Integration: ArgoCD notifications back to CI-CD Agent
